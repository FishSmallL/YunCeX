# 1. 401 Client Error: Unauthorized

先配置kaggle api

```bash
# 登入
kaggle auth login
# 配置api
kaggle auth print-access-token
```

将token添加到环境变量中

在Linux中可以放到.bashrc

在win可以放到系统环境变量

注，如果在win中还是报401的错，则用下面这条去临时设置

```
$env:KAGGLE_API_TOKEN="token"
```

如果在Linux也还是报错，则在终端export去设置

`export KAGGLE_API_TOKEN="token"`
